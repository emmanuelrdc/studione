# Carpeta de Imágenes para el Slideshow

Coloca aquí las imágenes que quieres que aparezcan como fondo en el Hero.

## Instrucciones:

1. Coloca tus imágenes en esta carpeta (`/public/place/`)
2. Nombra las imágenes como: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, etc.
3. El componente automáticamente las cargará y las cambiará cada 5 segundos

## Formato recomendado:

- Formato: JPG, PNG o WebP
- Tamaño recomendado: 1920x1080px o similar
- Las imágenes se mostrarán con 30% de opacidad como fondo

## Personalizar:

Si quieres cambiar el número de imágenes o los nombres, edita el archivo:
`/components/background-slideshow.tsx`
